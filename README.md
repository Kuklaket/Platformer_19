# Platformer_19

Внедрен разработанный ранее индикатор здоровья (плавный бар здоровья) в платформер. Над каждым врагом и игроком находиться его персональный индикатор здоровья, привязанный к нему.

Так же переработана система управления анимацией атаки. Вместо bool-параметра используется триггер (Trigger)

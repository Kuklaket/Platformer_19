# Platformer_19

0.1. Внедрен разработанный ранее индикатор здоровья (плавный бар здоровья) в платформер. Над каждым врагом и игроком находиться его персональный индикатор здоровья, привязанный к нему.
Так же переработана система управления анимацией атаки. Вместо bool-параметра используется триггер (Trigger) 

0.2.  Улучшена архитектура. 

В управление атаки врага (EnemyAttackHandler) добавили OnTriggerStay2D для реализации постоянной проверки наличия цели в зоне атаки каждый кадр.
Разделение ответственности - убрала из атакера зависимости от аниматора и врага, оставила только логику атаки, а так же создала общий базовый класс для атакеров игрока и врага, вынесла туда повторяющийся код.
